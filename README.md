# 💰 Personal Finance Tracker

A full-stack web application built with **Python Django** to manage personal expenses efficiently.

## 🌐 Live Demo
[Click here to view the live app](https://railway-up-production-1b0a.up.railway.app)

## ✨ Features
- 🔐 User Registration & Login
- ➕ Add Expenses with reason and amount
- ✏️ Update existing expenses
- 🗑️ Delete expenses
- 🔍 Search expenses by name
- 💵 Total spending summary on dashboard
- 📱 Responsive UI with Bootstrap 5

## 🛠️ Tech Stack
| Technology | Usage |
|---|---|
| Python | Backend logic |
| Django | Web framework |
| SQLite | Database |
| HTML/CSS | Frontend |
| Bootstrap 5 | Styling |
| Railway | Deployment |

## 🚀 Run Locally
```bash
git clone https://github.com/Dasarihemavathi/Personal-Finance-Tracker.git
cd Personal-Finance-Tracker
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
